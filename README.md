# Deepfake Image & Video Detector

An AI-powered system to detect deepfake images and videos using deep learning models.

## Features
- Image Deepfake Detection (CNN based)
- Video Deepfake Detection using frame-level analysis
- Streamlit-based interactive UI
- Confidence score for predictions

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
